# Portafolio de ejercicios de TypeScript  

Este repositorio contiene varios ejercicios y proyectos breves que he realizado para practicar TypeScript. Quise crear un lugar donde organizar lo aprendido y repasar conceptos cuando lo necesite.  

## Estructura  

- **arrays**: ejemplos de manipulación de listas y funciones sobre arreglos.  
- **infer**: ejercicios de inferencia de tipos usando la palabra clave `infer`.  
- **mappedTypes**: prácticas con tipos mapeados para generar tipos a partir de otros.  
- **narrowing**: ejemplos de estrechamiento de tipos y comprobaciones en tiempo de ejecución.  
- **poo**: clases y herencia para practicar programación orientada a objetos en TypeScript.  
- **transfDatos**: transformaciones de datos como filtrar y mapear estructuras.  
- **union**: ejercicios con tipos de unión y discriminación de tipos.  

Cada carpeta incluye uno o varios archivos `.ts` con el código comentado para explicar lo que se busca aprender. Espero que este portafolio te sirva como referencia personal o para otros que estén empezando con TypeScript.
